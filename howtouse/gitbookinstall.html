GitBook 使用入门

Gitbook安装

Gitbook是从NMP安装的，命令行：

$ npm install gitbook -g

安装完之后，你可以检验下是否安装成功：

$ gitbook -V
0.4.2

如果你看到了与上面类似的版本信息，则表示你已成功完装上了Gitbook。
